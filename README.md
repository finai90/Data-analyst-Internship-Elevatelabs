# Data-analyst-Internship-Elevatelabs
Data analyst Internship Elevatelabs
TASK 01
Coffee Beans Sales Data Cleaning & Preparation
Project Overview

This project demonstrates the data cleaning and preparation process using a Coffee Beans dataset downloaded from Kaggle. The objective was to transform raw data into a clean and structured dataset suitable for analysis and visualization.
Objective
Clean raw sales data
Improve data quality
Prepare the dataset for analysis and dashboard creation
🛠️ Tools Used
Microsoft Excel
Kaggle Dataset
Data Cleaning Process
Step 1: Data Collection
Downloaded the Coffee Beans dataset from Kaggle.
Imported the dataset into Microsoft Excel.
Step 2: Data Cleaning

Performed the following data cleaning tasks:

Removed duplicate records.
Identified and handled missing (null) values.
Corrected inconsistent data formatting.
Standardized text values where required.
Checked data types and ensured consistency.
Step 3: Data Preparation
Converted the cleaned data into an Excel Table.
Organized columns for better readability.
Created summary statistics to validate data quality.
Prepared the dataset for further analysis and visualization.
✅ Outcome
The final dataset is:
Clean
Consistent
Free from duplicate records
